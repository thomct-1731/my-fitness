# my-fitness
Get idea from [TDEE Calculator](https://tdeecalculator.net/) and using Django 4.1.2 to implement.

#### Not Login
1. Show foods list
2. Search by name food
3. Pagination foods

#### Logined
1. Show foods list
2. Search by name food
3. Pagination foods
4. CRUD a food
5. CRUD profile's user
5. Calculate TDEE
